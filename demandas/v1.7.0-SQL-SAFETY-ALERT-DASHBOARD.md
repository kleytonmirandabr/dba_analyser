# v1.7.0 — SQL Safety + Alert Dashboard + Query UX

**Data:** 2026-05-26

## Entregas

### 🛡️ Segurança SQL (Query Editor)
- **UPDATE sem WHERE** → BLOQUEADO (não executa)
- **DELETE sem WHERE** → BLOQUEADO (não executa)
- **WHERE 1=1 / WHERE TRUE** → BLOQUEADO (bypass detectado)
- **DROP DATABASE / TRUNCATE TABLE** → BLOQUEADO em produção
- **SELECT sem WHERE** em produção → Warning (executa com aviso)
- Mensagem clara no editor com ícone 🛡️ e sugestão de correção

### 🔍 Validação Real de Sintaxe SQL (Alertas)
- Ao criar/editar alerta: query é validada **no banco de destino** antes de salvar
- MSSQL: usa `SET NOEXEC ON` (parseia sem executar)
- PostgreSQL: usa `EXPLAIN` (valida plano)
- Impede cadastro de alertas com query quebrada

### 📊 Alert Dashboard (nova view)
- Timeline por hora: gráfico de área empilhado (OK/Disparados/Erros)
- Gráfico de valor escalar ao longo do tempo (para alertas tipo threshold)
- Stats consolidados: total checks, OK, alertas, erros, tempo médio de execução
- Toggle Dashboard / Lista
- Cards de resumo no topo (Total, OK, Disparados, Erro)

### ⏹️ Botão Stop na Query
- Botão vermelho "Parar (Xs)" substitui "Executar" durante execução
- AbortController cancela o request HTTP
- Contador de tempo decorrido em tempo real
- Mensagem "Execução cancelada pelo usuário" ao parar
