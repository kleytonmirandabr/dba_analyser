# v1.6.0 — Performance + Query IDE + UX Polish

**Data:** 2026-05-26

## Entregas

### 🚀 Performance (Critical Fix)
- Cache de chaves PBKDF2 na criptografia — `/api/connections` caiu de **~4.8s** para **<100ms**
- Problema: 100.000 iterações de PBKDF2 × 62 conexões × 2 decrypts = 4.8s por request
- Solução: LRU cache de chaves derivadas (mesmo salt = mesma key, PBKDF2 é determinístico)
